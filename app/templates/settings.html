{# ── How to get an App Password ──────────────────────────────────────────── #}
How to get a Gmail App Password
  1. Sign in to the Gmail account you want to use as the sender.
  2. Go to Google Account → Security → 2-Step Verification and make sure it's turned on.
  3. At the bottom of the 2-Step Verification page, click App passwords.
  4. Choose Other (Custom name), type VM Manager, and click Generate.
  5. Copy the 16-character code and paste it into the App Password field above.

You only need to do this once. The password does not expire unless you revoke it or disable 2-Step Verification.
